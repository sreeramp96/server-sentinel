# 📡 Server Sentinel

**Server Sentinel** is a clean, powerful, and modern website uptime monitoring tool built with **Laravel 12**, **Tailwind CSS v4**, and **Pest**. It provides real-time monitoring, response time tracking, and instant downtime alerts to ensure your digital assets stay healthy.

## ✨ Features

- 🕵️ **Real-time Monitoring**: Automatically check website health every minute.
- 📉 **Performance Metrics**: Track response times (ms) and status codes over time.
- 🔔 **Instant Alerts**: Receive email notifications the moment a site goes down.
- 📊 **Insightful Dashboard**: A beautiful interface showing uptime percentages, average response times, and history.
- 🔒 **Secure Authentication**: Built-in user management and profile settings powered by Laravel Breeze.
- 🧹 **Automatic Cleanup**: Daily logs cleanup to keep your database lean.

## 🏗 Tech Stack

- **Backend**: [Laravel 12](https://laravel.com) (PHP 8.2+)
- **Frontend**: [Tailwind CSS v4](https://tailwindcss.com) & [Alpine.js](https://alpinejs.dev)
- **HTTP Client**: [Guzzle](https://docs.guzzlephp.org/)
- **Testing**: [Pest PHP](https://pestphp.com)
- **Authentication**: [Laravel Breeze](https://laravel.com/docs/breeze)
- **Monitoring Logic**: Custom Artisan commands and Queued Jobs.
